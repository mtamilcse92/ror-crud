# ror-crud

Settup and document for [Web App](https://github.com/mtamilcse92/ror-crud/tree/main/web-app/README.md)

Settup and document for [Backend Service](https://github.com/mtamilcse92/ror-crud/blob/main/backend-service/README.md)

**User List page **

**Actions**
 - Create User -> will take to user create page and create new user
 - Edit Icon -> will take to user edit page and update the user
 - Delete Icon -> will directly delete the user
 - Total Order -> will take to user detail page (order details will be available)
